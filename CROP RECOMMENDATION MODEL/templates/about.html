Our Mission

Agriculture feeds the world, yet most smallholder farmers lack access to data-driven tools to receive crop recommendations in advance.

This platform bridges that gap by combining historical market records with machine learning models, delivering actionable crop recommendations through a clean and accessible web interface.

What is crop recommendation?

Crop recommendation uses environmental and input attributes to recommend the most suitable crop for a given field and season.

How this project is structured

  • Dataset explorer to inspect the exact training dataset rows.
  • Dashboard with trend, monthly, state and commodity visual analytics.
  • Prediction module with connected State, District, Market, Commodity and Variety workflow.
  • Model evaluation module for comparative train/test metrics.
  • Data preprocessing and EDA pages for transparency of pipeline steps.

Tech Stack

Frontend: HTML, CSS, JavaScript, Chart.js
Backend: Flask + Pandas + scikit-learn
Models: Crop price regression pipelines
